# 🚀 高考英语 3500 词消消乐 (Vocab Master PWA)

这是一个专门为高中生设计的英语单词高效记忆工具。通过“消消乐”的游戏化逻辑，结合艾宾浩斯遗忘曲线算法，帮助你在 70 天内攻克高考 3500 词。

## ✨ 核心亮点
- **消消乐玩法**：点击“已掌握”触发消除动画，连续正确触发 Combo 连击翻倍积分。
- **记忆算法**：内置简化版间隔重复算法（SRS），自动安排复习时间，确保长时记忆。
- **PWA 支持**：支持离线使用，可“添加到主屏幕”作为原生 App 使用，无须下载安装包。
- **极简设计**：Notion 风格 UI，专注学习，无广告干扰。

## 🛠️ 技术栈
- **前端框架**: React + Vite
- **本地存储**: Dexie.js (IndexedDB) - 即使离线，你的学习进度也不会丢失
- **动画效果**: Framer Motion
- **样式处理**: Tailwind CSS

## 📖 如何使用
1. **在线访问**: [点击这里访问预览链接](你的Vercel部署链接)
2. **离线安装**: 
   - iOS: 使用 Safari 打开链接，点击“分享” -> “添加到主屏幕”。
   - Android: 使用 Chrome 打开链接，点击右上角三个点 -> “安装应用”。
